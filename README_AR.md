# Protein & Carb Labels - Firestore Version

## التشغيل المحلي
افتح `index.html` مباشرة للتجربة.

## رفع Firebase Hosting
داخل مجلد المشروع شغل:

```bash
firebase login
firebase init hosting
firebase deploy
```

عند السؤال عن public directory اكتب:

```bash
.
```

## قاعدة البيانات
النظام ينشئ تلقائياً:
- products
- labelSettings/default

لا تضف الأصناف من لوحة Firebase. أضفها من داخل النظام.
